# Command-Line Expense Tracker

A fast, lightweight command-line application built in Java that allows users to seamlessly log daily expenses, categorize their spending, and save data persistently.

## Features
* **Interactive Menu:** Simple and intuitive text-based interface.
* **Add Expenses:** Log an expense with a date, category, and amount.
* **View History:** Display all previously recorded expenses in a clean format.
* **Persistent Storage:** Automatically saves and loads data from a local `expenses.txt` file using Java Stream I/O.

## Technologies Used
* Java (Standard Edition)
* File I/O (FileWriter, BufferedReader)

## How to Run
1. Make sure you have the Java Development Kit (JDK) installed on your system.
2. Clone this repository to your local machine.
3. Open your terminal and navigate to the project folder.
4. Compile the code by running:
   `javac ExpenseTracker.java`
5. Run the application by typing:
   `java ExpenseTracker`
